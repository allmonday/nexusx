# sqlmodel-nexus

[中文](./index.zh.md)

**sqlmodel-nexus** is a progressive SQLModel extension. Start from ORM entities, extend with non-ORM relationships, and generate GraphQL for interaction. Use `DefineSubset` to declaratively construct response DTOs.
